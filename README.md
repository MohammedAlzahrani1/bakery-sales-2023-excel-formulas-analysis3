# Bakery Sales Revenue Analysis – Excel Project

## Project Overview
This project is part of the Google Data Analytics Certificate (Course 2 – Unit 3).
It focuses on using Excel formulas to calculate total quantities sold and total
revenue for each bakery product.

## Tool Used
- Microsoft Excel

## Dataset Description
The dataset includes bakery sales data from March 25 to March 31, 2020.
It contains product names, prices, quantities sold, and calculated revenue.
The data was created for learning purposes.

## Tasks Completed
- Added a Revenue column using Excel formulas
- Calculated revenue per transaction
- Calculated total quantity sold per product
- Calculated total revenue per product
- Analyzed sales and revenue performance

## Skills Demonstrated
- Excel formulas and calculations
- Revenue and sales analysis
- Data summarization
- Business insight generation
